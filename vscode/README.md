# Roach Language Support

[![license](https://img.shields.io/github/license/LabMarket/roach.svg)](https://github.com/LabMarket/roach/blob/master/LICENSE)

Syntax highlighting and language support for the Roach language in VS Code.

## Features

*   Syntax highlighting for `.roach` files.
*   Language configuration for Roach (based on `language-configuration.json`, not directly included here).

## Installation

1.  Install the Roach Language Support extension from the VS Code Marketplace.
2.  Open a `.roach` file to activate the syntax highlighting.

## Usage

This extension provides syntax highlighting for the Roach language. See the [Roach Language Documentation](https://github.com/LabMarket/roach) for details on the language itself.
